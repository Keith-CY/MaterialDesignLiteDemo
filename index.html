Badge Configuration
MDL Class Effect Remarks
mdl-badge Defines badge as an MDL component Required on span or link
mdl-badge--no-background Applies open-circle effect to badge Optional
mdl-badge--overlap Make the badge overlap with its container Optional
data-badge='value' Assign string value to badge Not a class, but a seperate attribute; required on span or link

Button Configuration
MDL Class Effect Remarks
mdl-button Defines button as MLD component Required
mdl-js-button Assign basic MDL behavior to button Required
none Applies flat display effect to button(default)
md-button--raised Applies raised display effect Mutually exclusive with fab, mini-tab, and icon
mdl-button--fab Applies fab(circle) display effect Mutually exclusive with raised, mini-tab, and icon
mdl-button--mini-fab
mdl-button--icon Applies icon(small plain circular) display effect
mdl-button--colored Applies colored display effect(primary or accent color depending on the type of button) Colors are defined in material.min.css
mdl-button--primary Applies primary color display effect
mdl-button--accent Applies accent color display effect
mdl-js-ripple-effect Applies ripple click effect May be used in combination with any other classes
